# Main-screen redesign — design targets

Reference mockups for the `ui-redesign` of the Unplug main screen. Generated as
exploration aids (Gemini), so treat them as **intent, not spec** — exact spacing,
icons, and copy are illustrative.

| File | Role |
|------|------|
| `layout-target.jpg` | Overall layout & hierarchy: slim status banner → app-grid card → schedule → pinned CTAs. |
| `slider-target-inactive.jpg` | The dual-handle 24h schedule range slider + app grid, inactive state. |
| `slider-target-active.jpg` | Active/locked state: compact status line, locked slider with "now" marker, lock-badged app grid. |

Agreed direction: the layout from `layout-target` with the schedule replaced by
the dual-handle range slider shown in the `slider-target` shots.
